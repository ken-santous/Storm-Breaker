
### Installation On Kali Linux


```bash
$ git clone https://github.com/ultrasecurity/Storm-Breaker
$ cd Storm-Breaker
$ chmod +x install.sh
$ sudo ./install.sh  
$ python3 -m pip install -r requirments.txt
$ sudo bash install.sh 
$ sudo pip3 install pyngrok
$ sudo python3 Storm-Breaker.py
```
